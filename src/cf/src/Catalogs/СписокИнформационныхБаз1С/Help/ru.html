Справочник "Информационные базы 1С", используется для автоматического формирования списка информационных баз 1С:Предприятие (8.1/8.2/8.3) для пользователей сети.

Есть возможность делать резервные копии информационных баз работающих на MS SQL-сервере. Поддержка быстрого восстановления информационной базы в указанный приемник.

Настройка

Создать необходимые записи информационных баз 1С в справочнике.
Для настройки работы необходимо задать Список прав доступа (подразделение, пользователь) и в справочнике "Структура предприятия" установить флажок "Включено обновление списка Баз 1С" у всех необходимых для работы с 1С подразделений. Сформировать скрипты запуска пользователей и перезапустить компьютеры пользователей.
