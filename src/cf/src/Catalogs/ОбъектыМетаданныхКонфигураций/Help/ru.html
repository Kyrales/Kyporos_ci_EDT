Объекты метаданных - объекты относительно которых ведется учет версий конфигурации (принадлежат к определенной конфигурации). В табличной части "Изменения по умолчанию" задаются типы модулей, которые используются при формировании данных на основе старой версии конфигурации регистра "Учет версий конфигурации".

Табличную часть "Изменения по умолчанию" необходимо обновлять, если прошли изменения в обрабатываемой конфигурациии. Обновление табличной частие производится: вручную в табличной части или при использовании функции Импорт из регистра "Учет версий конфигураций", а также можно обновить с использованием кнопки "Записать изменения в ОМ" в регистре "Учет версии конфигурации".
