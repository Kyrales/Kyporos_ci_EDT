Справочник предназначен для использования в задачах системы. По проекту могут формироваться: отчеты, группировки, расчеты трудоемкости. Рекомендуется заполнять рабочую группу.

На странице "Шаблон задачи" можно заполнить типовую форму описания задачи (например: "Обновление версии 1С на ")

На странице "Рабочая группа проекта" указываются участники данного проекта. При выборе проекта в задаче, автоматически заполняется рабочая группа задачи (только справочники "Пользователи", "Организации", "Рабочие места")

Предопределенное значение "Избранные проекты" дает возможность составить выборку интересных проектов, по которым в списке "Мои задачи" можно делать интерактивный отбор.

Предопределенное значение "Без проекта" дает возможность отбирать задачи без привязанных проектов в списке "Мои задачи".
