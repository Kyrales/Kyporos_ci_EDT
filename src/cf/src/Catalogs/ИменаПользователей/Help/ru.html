Справочник используется в подсистемах: Менеджер журнала регистрации и Структура каталогов AD.

Для подсистемы "Менеджер журнала регистрации", справочник используется в качестве отражения данных пользователей из журналов регистрации других информационных баз.

Для подсистемы "Структура каталогов AD", справочник используется в качестве зеркального отражения пользователей AD и служит источником формирования сетевых каталогов Windows. Ключевой атрибут принадлежности к AD: Принадлежит к AD.
