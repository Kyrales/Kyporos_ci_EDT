Справочник "Шаблоны удаленного доступа"

Справочник предназначен для хранения информации о шаблонах подключения удаленного доступа.
Шаблон подключения удаленного доступа есть настройка параметризованной командной строки, для запуска программы удаленного доступа.

Реквизиты:

·        Наименование – Наименование шаблона.

·        Рабочий каталог – Рабочий каталог, в котором будут выполняться команды.

·        Шаблон командной строки – Шаблон командной строки, который при вызове удаленного подключения, будет заполнен из дополнительных свойств рабочего места.

Для простановки свойства в шаблоне необходимо указать ключ %Свойство%.

Например для удаленного рабочего стола:

Наименование = "Удаленный рабочий стол" 

Рабочий каталог = "c:\windows\system32\"

Шаблон командной строки = "mstsc.exe /V:%Имя или IP%"

У рабочего места должно присутствовать дополнительное свойство "Имя или IP"  в значении которого должен быть указан конкретный IP адрес или сетевое имя компьютера к которому будет осуществляться подключения.

Справочник доступен из командного интерфейса подсистемы "Администрирование".
