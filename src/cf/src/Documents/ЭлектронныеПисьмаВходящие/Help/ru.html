Документ "Электронные письма входящие"

Документ "Электронное письмо входящее" предназначен для регистрации полученных электронных писем. При получении нового входящего электронного письма пользователь может выполнить следующие действия.  

Для включения загрузки входящих электронных писем, необходимо настроить учетную запись электронной почты в режиме "Для получения писем". И установить признак "Использовать для загрузки входящих электронных писем" на закладке "Прочие настройки".

Регламентные задание, которые обеспечивают механизм регистрации задач по электронным письмам:
- Получение электронных сообщений
- Обработка электронных сообщений
