Акты выполненных работ закрывают данные документов "Работы по задаче" по выставленным часам.

- Заполнение выполненных задач по критериями: за период даты выполнения, все задачи. В момент заполнения исключать задачи, которые уже были закрыты по другим актам (вручную добавлять их можно).
- Учет и расчет суммы из трудоемкости задач
- Формирование печатной формы "Счет на оплату", "Акт выполненных работ"
- Учитывать: признак "Оплачено", Сумма оплаты, Дата оплаты.

Проведение

Движения формируются по дате документа. Не важно Оплачен ли акт или нет. При проведение проходит контроль остатков трудоемкости

Если будет обнаружено меньше трудоемкости чем закрывается, то будет выдана ошибка
