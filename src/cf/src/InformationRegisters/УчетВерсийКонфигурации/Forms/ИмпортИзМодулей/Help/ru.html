Используетя для импорта и обновления информации из модулей об изменениях нетиповых конфигураций.

Инструкция по работе с импортом данных

1. В конфигураторе необходимой конфигурации: Конфигурация -> Выгрузить файлы конфигурации и убираем все галочки кроме "Модули"

2. Для начала загрузки в "Купорос": в поле ввода "Конфигурация" выбираем необходимую и далее кнопка "Импорт модулей". После нажатия, выбираем каталог в который выгрузили данные о модулях конфигурации

3. После успешной загрузки корректируем строки где не определился тип объекта метаданных или тип модуля

4. Отмечаем необходимые для обновления данные флажками "Для обновления" и нажимаем кнопку "Обновить данные объектов метаданных" 

Внимание! Импорт работает только там, где используется оформление с использованием методологии "Купорос"
