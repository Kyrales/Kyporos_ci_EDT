Подсистема используется для формирования гибкой структуры папок Windows на основе данных о пользователях Active Directory предприятия в автоматическом режиме.

Системные администраторы в Active Directory (AD) добавляют/изменяют запись пользователя, а подсистема "Структура каталогов AD" на основе этих изменений осуществляет автоматический анализ и изменение сетевых папок, зависимых от: подразделения пользователя, должности пользователя, ФИО пользователя и других параметров. После изменения сетевых папок осуществляется пересчет прав на данную сетевую папку (также учитываются права директоров/ИТ-специалистов).
В случае если запись пользователя AD отключили, то сетевая папка пользователя будет автоматически перемещена в папку "История предприятия" и все права по ней обнулены.

Подсистема "Структура каталогов AD" является заменой использования сложных скриптов, для повышения удобства работы с каталогами пользователей сети предприятия. Автоматизируется процесс перемещения каталогов пользователей в папку уволенных или в другое подразделение. Расширенный анализ и контроль квот на каталоги пользователей.
