Обработка, используется для формирования шапки модуля объекта (по переменным, функциям и процедурам) на базе методологии "Купорос".

Для формирования шапки модуля объекта необходимо скопировать в поле ввода "Текст модуля объекта" текст из модуля объекта, на основании которого необходимо подготовить шапку модуля объекта.
